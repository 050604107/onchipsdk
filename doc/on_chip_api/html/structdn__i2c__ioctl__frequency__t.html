Detailed Description

Specify a new frequency setting.

Pass (a pointer to) this structure in the args parameter of dn_open(), when setting request to DN_IOCTL_I2C_SETFREQ.

Field Documentation

frequency

I2C clock frequency.

The following frequencies settings are available:

  • DN_I2C_FREQ_184_KHZ
  • DN_I2C_FREQ_123_KHZ
  • DN_I2C_FREQ_92_KHZ
