loadBattery

Load resistors configuration for battery test.

The LTC5800 features battery load resistors which can be enabled when measuring the supply voltage.

Measuring the voltage of a battery under different loads can help your application assess the health of your batteries.

See Battery.

Acceptable value are:

  • DN_ADC_LOAD_BATT_NONE
  • DN_ADC_LOAD_BATT_100_OHM
  • DN_ADC_LOAD_BATT_500_OHM
  • DN_ADC_LOAD_BATT_83_OHM
