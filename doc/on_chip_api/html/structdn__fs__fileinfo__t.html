Field Documentation

mode

Mode of the file.

This is a bitmap which can contain the following flags:

  • DN_FS_MODE_OTH_R
  • DN_FS_MODE_OTH_W
  • DN_FS_MODE_OTH_RW
  • DN_FS_MODE_TEMP
  • DN_FS_MODE_SHADOW
