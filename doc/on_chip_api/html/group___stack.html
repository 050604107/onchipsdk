Description

Drive the mote's communication stack.

Communication with the network stack happens through a (set of) Channel. The set of commands, responses and notifications your application can send to the network stack is detailed in the Raw Local Interface.

To simplify your implementation, you can use the Local Interface Module which provides a "wrapper" around the Raw Local Interface. When using the Local Interface Module, your code calls functions, which the Local Interface Module translates into the appropriate bytes sent over the Raw Local Interface channel.
