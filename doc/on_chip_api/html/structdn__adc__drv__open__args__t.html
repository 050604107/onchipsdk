Field Documentation

rdacOffset

Setting for the RDAC.

Note
Refer to the On-Chip Firmware Development Guide for a description of the RDAC.

Acceptable value are:

  • DN_ADC_RDAC_113mV
  • DN_ADC_RDAC_225mV
  • DN_ADC_RDAC_338mV
  • DN_ADC_RDAC_450mV
  • DN_ADC_RDAC_563mV
  • DN_ADC_RDAC_675mV
  • DN_ADC_RDAC_788mV
  • DN_ADC_RDAC_900mV
  • DN_ADC_RDAC_1013mV
  • DN_ADC_RDAC_1125mV
  • DN_ADC_RDAC_1238mV
  • DN_ADC_RDAC_1350mV
  • DN_ADC_RDAC_1463mV
  • DN_ADC_RDAC_1575mV
  • DN_ADC_RDAC_1688mV
  • DN_ADC_RDAC_1800mV
vgaGain

Setting for the VGA.

Note
Refer to the On-Chip Firmware Development Guide for a description of the VGA.

Acceptable value are:

  • DN_ADC_VGA_1800mVpp
  • DN_ADC_VGA_900mVpp
  • DN_ADC_VGA_600mVpp
  • DN_ADC_VGA_450mVpp
  • DN_ADC_VGA_360mVpp
  • DN_ADC_VGA_300mVpp
  • DN_ADC_VGA_257mVpp
  • DN_ADC_VGA_225mVpp
fBypassVga

If set, bypass (do not use) the RDAC and VGA.
